## 📂 Directory Structure

```
├── .github # github actions 파일
├── api  # API 서버
├── batch  # Batch 서버
├── client # 외부 api 호출 모듈(Open API, FCM, OAuth)
├── common # 공통 모듈(util, error...)
├── docker # 로컬 도커 환경 설정
├── domain # 도메인 모듈(domain, repository)
│   └── core # JPA + MySQL 모듈
│   └── redis # Redis 모듈
├── monitoring # 모니터링(prometheus, grafana, loki)
```
